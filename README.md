# How execute the project

## Back-End

Prerequisite: Java 17.

```bash

 git clone https://github.com/FelipeSdsilva/dscatalog
 
 #enter in folder backend.
 cd backend
 
 #execute the project.
 ./mvnw spring-boot:run
```
